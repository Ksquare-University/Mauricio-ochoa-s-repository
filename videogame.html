Do it Yourself

How to make it

This is just an overview on how to make your own videogame for the whole Do it yourself you can doownload the whole PDF, if not available the PDF is not ready or version 2 is coming.

What you need

you will need a lot of things to make this project, but they are easily available you will need:

  • • 3D printer or a case
  • • Design software
  • • Heat gun for joints
  • • 12 push button
  • • raspberry pi zero w
  • • LCD screen 240X320 pixels by Adafruit
  • • 24 caliber soft cable
  • • PCB for push buttons or the gerber files so you can do the PCB
  • • SD card
  • • Li-On battery with an output of 5-9V
  • • Charger module with Micro USB
  • • Switch

    • you will need to solder the push buttons into the custom PCB for the controller set Once this step is done you will need to solder the cable from the PCB to the GPIO ports of the Raspberry Pi (you can follow the diagram in the files below ) then do the same with the screen connect it to the Raspberry, however do not get to excited as you need to do some extra steps here so the image can be displayed on the LCD screen you will need to configure the Raspberry Pi using SSH connection or another screen to configure the Raspberry Pi so the output of the image is not the default (HDMI port) and change it to the GPIO ports where our LCD is connected.

    After setting up the screen it is necessary to download the Retropi image on the SD card this is where videogames are going to be played
