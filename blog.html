VIDEO GAME CONSOLE

DESCRIPTION

This project uses Raspberry Pi zero W to run an image for video games called Retro-pi.

This project is a videogame console that involves 3D printing, electronics and software skill, and knowledge with Raspberry pi, (at least worked with it before). Also you will need knowledge of some basic electronics concepts as well as knowledge soldering.

This project will use a 3D printer, so it is required to have one or contact me if you want to print own version of the videogame console.
As well, you can use a personalized PCB(printed circuit board) for the controls of the console, however you can follow this instructions without the PCB however I recommend you to do it or to contact me for one.
